Demonstrated Features:

  • BT.cameraSet(offset) - Set camera position for scrolling
  • BT.cameraGet() - Get current camera offset
  • BT.cameraReset() - Reset camera to origin (used for the UI)
  • Large world (800×600) with viewport scrolling (320×240)
  • Mini-map showing camera position in world
  • UI overlay that doesn't scroll with camera
Note: The camera auto-scrolls in a pattern since input is not yet implemented in Phase 1. Future versions will allow manual camera control with a keyboard/mouse.
