Animated Patterns:

This demo showcases various mathematical patterns and visual effects created using primitive drawing functions.

Spiral Parametric spiral with color gradient
Radial Lines Pulsing rays from center point
Wave Multiple sine waves with interference
Circle Polygon approximation with rotation
Lissajous Complex parametric curve (3:4 ratio)
Tunnel Concentric rectangles with offset

All patterns are rendered in real-time using drawPixel(), drawLine(), and drawRect().
