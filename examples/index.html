Phase 1 Features Implemented:

  • Core type system: Vector2i, Rect2i, Color32
  • WebGPU dual-pipeline renderer (primitives + sprites)
  • Primitive rendering: pixels, lines, rectangles
  • Sprite rendering with texture batching and color tinting
  • Bitmap font system with fixed-width font support
  • Asset loading: SpriteSheet, BitmapFont
  • Camera system with offset and reset
  • Fixed timestep game loop (60 FPS)
  • Clear operations (full screen and rectangular regions)

Not Yet Implemented (Planned):

  • Keyboard input (WASD, Arrow Keys) - BT.keyDown(), BT.keyPressed()
  • Gamepad support - BT.buttonDown()
  • Mouse input
  • Palette system (indexed colors, color cycling)
  • Tilemap rendering
  • Audio playback

Requirements:

  • Chrome/Edge 113+ or Firefox Nightly
  • WebGPU support enabled
  • Compatible GPU and drivers
